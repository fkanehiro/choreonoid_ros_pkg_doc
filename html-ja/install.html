インストール¶

概要¶

本プラグインは ChoreonoidのROS接続機能を提供します。

本プラグインは以下のパッケージで構成されています。

  • choreonoid_ros_pkg: Choreonoid関連パッケージをビルドするためのメタパッケージ

  • choreonoid_ros: Choreonoid本体のcatkinパッケージ

  • choreonoid_plugin: ROSトピックをpublishするためのChoreonoidプラグイン

  • jvrc_models: JVRC競技会用シミュレーションモデルとタスクファイル

動作環境¶

本プラグインは以下の環境で動作する事を確認しています。

  • Ubuntu 14.04 64-bit PC (AMD) desktop + ROS Indigo

OS のインストールは事前に完了しておいてください。

OS のインストール方法については http://www.ubuntu.com/download/desktop/install-ubuntu-desktop を参照してください。

ROS および、ビルド時に必須となるパッケージのインストール方法は、次節で説明します。

インストール前準備¶

ROS および、ビルド時に必須となるパッケージのインストール方法を以下に記します。

ROS Indigo のインストール¶

以下のコマンドを実行します:

$ sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'
$ sudo apt-key adv --keyserver hkp://ha.pool.sks-keyservers.net --recv-key 0xB01FA116
$ sudo apt-get update
$ sudo apt-get install ros-indigo-desktop-full
$ echo "source /opt/ros/indigo/setup.bash" >> ~/.bashrc
$ source ~/.bashrc

手順の詳細については http://wiki.ros.org/indigo/Installation/Ubuntu を参照してください。

必須パッケージのインストール (simtrans 以外)¶

以下のコマンドを実行します:

$ sudo apt-get install python-rosinstall python-catkin-tools

インストール後、ROS のパッケージデータベースの初期化および更新を実行します:

$ sudo rosdep init
$ rosdep update

以下のコマンドを実行し、エラーが発生しない事を確認します:

$ rosdep db
          :
        <snip>
          :
$ echo $?
0

rosdep コマンドの詳細については http://wiki.ros.org/rosdep を参照してください。

必須パッケージのインストール (simtrans)¶

以下のコマンドを実行します:

$ sudo add-apt-repository ppa:hrg/daily
$ sudo apt-get update
$ sudo apt-get install python-pip openhrp meshlab imagemagick python-omniorb openrtm-aist-python
$ git clone https://github.com/fkanehiro/simtrans.git
$ cd simtrans
$ sudo pip install -r requirements.txt
$ sudo python setup.py install

インストールの確認をします:

$ which simtrans
/usr/local/bin/simtrans
$ simtrans -h
usage: simtrans [-h] [-i FILE] [-o FILE] [-f FORMAT] [-c] [-b] [-t FORMAT]
                [-p PREFIX] [-s] [-e SPGR] [-v]

Convert robot simulation model from one another.
          :
        <snip>
          :
$ echo $?
0

simtrans コマンドの詳細については https://github.com/fkanehiro/simtrans を参照してください。

Choreonoid ROS プラグインのインストール¶

Choreonoid ROS plugin のインストール方法を以下に記します。

パッケージの利用に際し、catkin ワークスペースを作る必要があります:

$ mkdir -p ~/catkin_ws/src
$ cd ~/catkin_ws
$ catkin init

catkin_ws/srcフォルダ以下にchoreonoid_ros_pkgをチェックアウトします:

$ cd ~/catkin_ws/src
$ wstool init
$ wstool set choreonoid_ros_pkg https://github.com/fkanehiro/choreonoid_ros_pkg.git --git -y
$ wstool update choreonoid_ros_pkg

依存パッケージをインストールします:

$ cd ~/catkin_ws
$ rosdep install --from-paths src --ignore-src --rosdistro $ROS_DISTRO -y

catkinパッケージをビルドしインストールします:

$ cd ~/catkin_ws
$ export CMAKE_PREFIX_PATH=~/catkin_ws/devel:/opt/ros/indigo
$ catkin config --install
$ catkin build choreonoid_ros_pkg
$ source install/setup.bash

URDF/SDF形式のモデルをChoreonoidで使用する場合、sdfloaderを追加でインストールします:

$ cd ~/catkin_ws/src
$ wstool set choreonoid_sdfloader_plugin https://github.com/fkanehiro/choreonoid-sdfloader-plugin.git --git -y
$ wstool update choreonoid_sdfloader_plugin
$ cd ~/catkin_ws
$ catkin build choreonoid_sdfloader_plugin

ROS のデフォルトセットアップを変更します。

任意のテキストエディタで ~/.bashrc の’source /opt/ros/indigo/setup.bash’の行を以下のように変更します:

source ~/catkin_ws/install/setup.bash

  • ROS のデフォルトセットアップに戻す場合。

    任意のテキストエディタで ~/.bashrc の’source ~/catkin_ws/install/setup.bash’の行を以下のように変更します:

    source /opt/ros/indigo/setup.bash

    変更後、以下のコマンドを実行します。

    $ source ~/.bashrc

Choreonoid ROS プラグインのアップデート¶

Choreonoid ROS plugin の更新方法を以下に記します。

以下のコマンドを実行します:

$ cd ~/catkin_ws/src
$ wstool update choreonoid_ros_pkg
$ wstool update choreonoid_sdfloader_plugin (*)
$ cd ~/catkin_ws
$ catkin clean -b
$ catkin build choreonoid_ros_pkg
$ catkin build choreonoid_sdfloader_plugin (*)
$ source install/setup.bash

(*) sdfloader をインストールした場合のみ実行します。

トラブルシュート¶

インストール後に問題が発生した場合の解決方法を以下に記します。

  • ‘roslaunch choreonoid_ros jvrc-1-rviz.launch’の起動において問題が発生した場合

    catkin の設定’Extending’の値を確認します:

    $ cd ~/catkin_ws
$ catkin config
           :
         <snip>
           :
Extending:             [env] /opt/ros/indigo
           :
         <snip>
           :

    • 値が’[env | cached] <path to your home directory>/catkin_ws/devel:/opt/ros/indigo’の場合:

      以下のコマンドを実行します:

      $ source install/setup.bash
$ roslaunch choreonoid_ros jvrc-1-rviz.launch

    • 値が’[env] /opt/ros/indigo’の場合:

      以下のコマンドを実行します:

      $ export CMAKE_PREFIX_PATH=~/catkin_ws/devel:/opt/ros/indigo
$ source install/setup.bash
$ roslaunch choreonoid_ros jvrc-1-rviz.launch

    • 値が’[cached] /opt/ros/indigo’もしくは、これまでのケースに合致しない場合:

      以下のコマンドを実行します:

      $ catkin clean -a
$ export CMAKE_PREFIX_PATH=~/catkin_ws/devel:/opt/ros/indigo
$ catkin build choreonoid_ros_pkg
$ catkin build choreonoid_sdfloader_plugin (*)
$ source install/setup.bash
$ roslaunch choreonoid_ros jvrc-1-rviz.launch

      (*) sdfloader を使用する場合のみ実行します。
