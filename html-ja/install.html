インストール¶

本レポジトリは ChoreonoidのROS接続機能を提供します。

choreonoid_ros_pkg: catkinパッケージ化されたChoreonoid

choreonoid_plugin: ROSトピックをpublishするためのChoreonoidプラグイン

インストール作業を始める前にwstoolとcatkinがインストールされていることを確認してください。

$ sudo apt-get install python-wstool python-catkin-tools

パッケージを利用するには、まずcatkinワークスペースを作る必要があります:

$ mkdir -p ~/catkin_ws/src
$ cd ~/catkin_ws/src
$ catkin_init_workspace
$ cd ~/catkin_ws
$ catkin_make
$ source devel/setup.bash

catkin_ws/srcフォルダ以下にchoreonoid_ros_pkgをチェックアウトします:

$ cd ~/catkin_ws/src
$ wstool init
$ wstool set choreonoid_ros_pkg https://github.com/fkanehiro/choreonoid_ros_pkg.git --git
$ wstool update choreonoid_ros_pkg

catkinパッケージをビルドします:

$ cd ~/catkin_ws
$ rm -rf build
$ catkin build choreonoid_ros
$ catkin build choreonoid_plugins
